# Сайт компании ООО «Энерго Эффект»

Описание компании

## 1. Добавление странцы на сайт

Скопировать директорию:

```
./archetypes/0_to_copy_news
```

### Новости

в директорию:

```
./content/news
```

### Партнеров

в директорию:

```
./content/partner
```

### Описание структуры страницы:

```
archetypes
├── -1_to_copy_news         <- Шаблон структуры для странц новстей/проектов
│   ├── images              <- Фото для слайдов (выводиться в алфовитном порядке по названию)
│   ├── index.md            <- Контент страницы
│   └── preview.webp        <- Превью страницы
└── default.md
```

## 2. Необходимые правки после копирования

- Изменить навание директори под страницу (этто будет URL страницы)
- Поменять `preview.webp`
- Изменить дату title в файле `index.md`
- Добавить фото для слайда в директорию `images` (при необходимости)
